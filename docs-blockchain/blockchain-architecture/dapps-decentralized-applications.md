# DApps: Decentralized Applications

## `DApps`: Decentralized Applications

> applications built on blockchain platforms that **leverage smart contracts** for various functionalities
>
> used on networks with **trustless protocols** to **avoid a single point of failure**

* there's no master copy of DApp
* not governed by central server
* open-source



**Advantages:**

* improved stability
* no downtime
* higher reliability&#x20;
* no third-party integration
* automatic credentials assigned
* privacy and security
