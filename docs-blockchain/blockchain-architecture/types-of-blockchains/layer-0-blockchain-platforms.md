# Blockchain Layers: Layer 0

## Layer 0 Platforms

> **Layer 0 Platforms:**
>
> * create a **network of blockchains**&#x20;
> * connects different blockchain networks together
> * include protocols which enable **`cross-chain operations`**

* ✅ used when we need **cross-network**, **cross-platform** compatibility



#### The "**`Internet of Blockchains`**"

* internet aimed to create ➝ a network of networks

similarly,&#x20;

* layer 0 platforms aim to create ➝ a network of blockchain networks
