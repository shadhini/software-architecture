# Web 2.0

## Web 2.0

> Web 2.0 →  the **second generation of the World Wide Web**
>
> "<kbd>**Read-Write Web**</kbd>": users could consume & produce content both
>
>
>
> **characteristics**
>
> * more interactive, active participatory, collaborative, user-driven online experience&#x20;
>
> **laid the foundation for**&#x20;
>
> * the social media landscape,&#x20;
> * the sharing economy, and&#x20;
> * the principles of user-generated content that continue to shape the modern internet



Web 2.0 represents a significant departure from the earlier Web 1.0



### Web 2.0 key features & characteristics

* User-Generated Content
  * blogs, social media, wikis, ...
* Interactivity and Collaboration
  * Users could&#x20;
    * comment on articles
    * share opinions&#x20;
    * engage in discussions&#x20;
  * led to the formation of online communities
* Social Networking
  * platforms:  Facebook, Twitter, Linkedin&#x20;
* Relies heavily on centralized servers and platforms



### What is Web 2.0 ?

* Rich Internet Applications / User Interfaces:
  * ﻿﻿ **`AJAX`** (**Asynchronous JavaScript and XML**) — enable the creation of more responsive and dynamic web applications — enable real-time updates on web pages without requiring a full page reload
  * real-time updates on web pages
  * ﻿﻿dynamic and interactive user interfaces
  * web applications feel more responsive
* Tagging and Folksonomies:
  * tagging: users could assign keywords or tags to content, making it easier to organize and search for information
  * **folksonomy**: this user-generated categorization system; taxonomies
* Content Sharing and User-Created Media:
  * platforms: YouTube, Flickr, Vimeo
  * users can upload, share, and view multimedia content&#x20;
  * democratized media creation and distribution
* User-Centric Design:
  * prioritize user experience, ease of use, intuitive interfaces, usability, accessibility, responsive design to accommodate various devices and screen sizes
  * more user-focused&#x20;
* **`APls`**(Application Programming Interfaces) and Mashups:
  * AP|s allow developers to create applications that interacted with external services and data sources and create mashups-web applications that combine data or functionality from multiple sources
* Crowdsourcing:
  * crowdsourcing: a large group of users contributes ideas, content, or solutions to a particular problem or project
* Collaboration and Participation:
  * users could create, edit, and share content on platforms&#x20;
    * wikis, blogs, social media platforms
  * "user-generated content" —  individuals are both consumers and producers of information
* Social Media and Networking:
  * platforms: Facebook, Twitter (X),  Linkedin
  * allow users to create profiles, share updates, connect with friends and colleagues, and build online communities
* Blogs and Microblogging:
  * Microblogging platforms: Twitter —  brief messages and updates in real-time
* Web Applications:
  * web applications that could mimic the functionality of traditional desktop software
  * provided tools and services online
* Democratization of Information:
  * individuals can contribute to the creation and dissemination of information
    * anyone could share their expertise, opinions, insights



### Web 2.0 Limitations

* centralized control
* data privacy concerns&#x20;
* reliance on fixed content frameworks
