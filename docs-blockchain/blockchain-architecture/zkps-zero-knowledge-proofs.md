# ZKPs: Zero Knowledge Proofs

## `ZKP`s: Zero Knowledge Proofs

> a method that **allows one party to prove to another that they know a value without revealing the value itself**



Example: prove that you know the password of a lock without revealing the password

* if you can you up to the lock and open it yourself, then you can prove that you indeed know the password, without revealing the password
