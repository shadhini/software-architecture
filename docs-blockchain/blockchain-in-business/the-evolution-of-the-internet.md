# The Evolution of the Internet

## The Evolution of the Internet



{% stepper %}
{% step %}
### Web 1.0

similar to **digital library**: where you could look, but not touch — interaction was limited to clicking hyperlinks and reading text

Characteristics

* "<kbd>**Read-Only Web**</kbd>"
* static HTML pages:  static content — most content was created and published by professionals
* little to no user interaction or dynamic content
* passive consumption of content

Key Features

* HTML-based
* Hyperlinks
* limited multimedia via GIFs and JPEGs
* one-way information flow

Use Cases

* digital brochures
* informational websites&#x20;
* basic online directories
{% endstep %}

{% step %}
### Web 2.0

transformed users from passive consumers into active contributors

similar to **digital town square** — buzzing, dynamic, interactive

Characteristics

* "<kbd>**Read-Write Web**</kbd>": users could consume & produce content both

Key Features

* Blogs&#x20;
* Wikis
* Social media platforms
* AJAX-based interactivity
* User-generated content

Use Cases

* social networking
* e-commerce
* online collaboration
* content sharing
{% endstep %}

{% step %}
### Web 2.5

a **bridge** between Web 2.0 and Web 3.0

Characteristics

* ongoing transition from Web 2.0 to Web 3.0
* involves integrating&#x20;
  * elements of decentralization and semantic understing&#x20;
  * but without fully embracing the decentralization aspect

Key Features

* Hybrid centralized-decentralized architecture
* limited use of smart contracts
* APIs for machine learning

Use Cases

* decentralized finance platforms with centralized interfaces
* transitional social media platforms
* enhanced search algorithms
{% endstep %}

{% step %}
### Web 3.0

similar to a **global brain** — a network of interlinked, machine-readable data that can be easily shared and processed across various platforms, all without central authorities

Characteristics

* "<kbd>**Semantic Web**</kbd>"
* aims for a more open, secure, and decentralized infrastructure

Key Features

* Blockchain technology
* **`dApps`**: Decentralized applications&#x20;
* Smart contract
* Data interoperability
* AI-driven

Use Cases

* Digital brochures&#x20;
* Informational websites
* Decentralized identity management
* open-source decentralized finance
* NFT marketplaces
* **`DAOs`**: Decentralized autonomous organizations
{% endstep %}
{% endstepper %}
