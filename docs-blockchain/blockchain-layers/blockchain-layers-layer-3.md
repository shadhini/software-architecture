# Blockchain Layers: Layer 3

## Layer 3 Platforms&#x20;

> **Layer 3 Platforms**:
>
> * built on Layer 2&#x20;
> * the application layer
> * supports&#x20;
>   * programming interfaces
>   * frameworks
>   * scripts for automation
>   * UI/UX components

* the newest, the most recent as of 2025

#### Metaphor: with respect to PC

OS ➝ Layer 1

API ➝ Layer 2

Application Layer (where you are actually building) ➝ Layer 3&#x20;



#### Notable Layer 3 Platforms

* `ILP`: Interledger Protocol (used by Ripple/XRP cryptocurrency)
* ICON
* Quant
* IBC Protocol (used by Cosmos)
