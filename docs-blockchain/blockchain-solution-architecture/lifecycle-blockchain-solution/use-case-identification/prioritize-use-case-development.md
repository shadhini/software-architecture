# Prioritize Use Case Development

## Prioritize Use Case Development

Once you've identified 1-3 good use cases:

1. Document each potential use case
2. Demonstrate support for one or more business objectives
3. Demonstrate a positive difference in the customer experience

#### Prioritize use case development

{% hint style="success" %}
﻿﻿**HINT**: Construct a <mark style="color:blue;">**scorecard**</mark>...
{% endhint %}

* ﻿﻿Which delivers the most business value? — expected value/ value delivered
* ﻿﻿Which creates the most business pain? — pain points
* Which would be the simplest problem to solve? — development complexity
* Which would introduce the most risk? — risks
