# Proof of Importance

## Proof of Importance

> **Proof of Importance**:
>
> creating blocks and validating transactions handled by **"important" nodes** to the network
>
> **Importance** = number of **coins/tokens held** + **transaction** **activity**&#x20;
>
> &#x20;                               \+ number of **coins sent/received**

* used by `NEM` blockchain&#x20;
* designed to **incentivize network use** and **discourage coin hoarding** (practice of accumulating and storing a large quantity of coins)
