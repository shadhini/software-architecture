# Ethereum Development Environment

## Ethereum Development Environment



#### Remix IDE

> A **browser-based compiler and IDE** that enables users to debug transactions and **build Ethereum contracts** with Solidity language

<figure><img src="../../.gitbook/assets/ethereum-REMIX-IDE.svg" alt="" width="563"><figcaption></figcaption></figure>



#### Truffle

> a **development environment**, **testing framework**, and **asset pipeline** for blockchains using the Ethereum Virtual Machine (**EVM**)

* Smart contract lifecycle management
* Automated contract testing
* Scriptable deployment and migrations
* Powerful interactive console

<figure><img src="../../.gitbook/assets/ethereum-truffle.svg" alt="" width="375"><figcaption></figcaption></figure>





#### Embark

> An **environment** that allows you to easily **develop and deploy decentralized applications**

* Performs automatic deployment of contracts
* Keeps track of deployed contracts
* Easily manages complex systems of interdependent contracts
* Performs test driven deployment with contracts using JavaScript
* Manages different chains like `testnet`, `private net`, and `livenet`
* Redeploys the contracts if any changes are done
