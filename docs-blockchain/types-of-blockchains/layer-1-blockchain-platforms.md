# Blockchain Layers: Layer 1

## Layer 1 Platforms

> **Layer 1 Platforms**: provide all the basic, low level functionality we expect from a blockchain
>
> provide & manage a ledger and&#x20;
>
> perform&#x20;
>
> * group consensus activities
> * resolve disputes&#x20;
> * implement basic network functionality such as&#x20;
>   * Smart Contract deployment & execution

{% hint style="info" %}
The **very first blockchain platforms** to be released were layer 1 platforms.
{% endhint %}

{% hint style="info" %}
Layer 1 blockchains&#x20;

act as the **foundation for Layer 2 and Layer 3 platforms**
{% endhint %}



#### Metaphor: with respect to PC

<kbd>Operating System ➝ Layer 1 blockchain</kbd>

* Layer 1 blockchains act as the operating system component in a blockchain solution



#### Popular Layer 1 Blockchains

* Bitcoin
* Ethereum
* Solana
* Hyperledger — permissioned layer 1 platform&#x20;
