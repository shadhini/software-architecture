**public key cryptography**

* method used to secure transactions & data on the blockchain&#x20;
* leverages 2 cryptographic keys: a **`public key`** and a **`private key`**
  * public key: allows others to send funds/data to the correct address
  * private key: confidential key that grants control over the associated digital asset
    * private key holder can authorize transactions verifying ownership
