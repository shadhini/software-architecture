# Encoding, Hashing, Encryption, Cryptography, Steganography, Tokenization, Data Masking

## Encoding vs Hashing vs Encryption vs Cryptography

#### Encoding&#x20;

transforms data for usability/compatibility (reversible, no security)&#x20;

✅ helps transfer data

`Base64` &#x20;

#### Hashing&#x20;

creates a unique, fixed-size fingerprint for integrity checks (irreversible, no key)&#x20;

✅ verifies data hasn't changed (passwords)

`SHA-256`

#### Encryption&#x20;

scrambles data (plaintext to ciphertext) using algorithms and keys for confidentiality (reversible with a key)

✅ keeps secrets safe

`AES`&#x20;

#### Cryptography

broad science of securing information by converting it into an **unreadable format**





## Steganography vs Tokenization vs Data Masking

#### Steganography&#x20;

hiding information within ordinary data like images, audio, video files

#### Tokenization&#x20;

replaces sensitive information with unique identifiers or tokens

#### Data Masking

masks/changes/replaces real sensitive data with fake, but realistic random data  &#x20;

for testing or analysis
