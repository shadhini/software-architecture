# File Encryption

## File Encryption

> encrypts individual files,documents

* enable highly selective encryption
* used in document sharing and backup
* specially useful for protecting documents with sensitive information

`Axcrypt` | `VeraCrypt`&#x20;



### Mode of Operation

* encrypts individual files with a unique key&#x20;
* requires a password or decryption key to aaccess and encrypted file
* file remains secure no matter where it is stored



### Strengths

* enable selective encryption
* reduces encryption overhead&#x20;
* offers portability across devices
  * files remain encrypted when moved to other devices or storage locations; USB drives, cloud storage&#x20;
* supports secure sharing&#x20;
  *   encrypted files can be shared using the decryption key&#x20;



### Limitations

* requires manual processing&#x20;
  * requires users to identify and encrypt files individually
* lacks full disk security
