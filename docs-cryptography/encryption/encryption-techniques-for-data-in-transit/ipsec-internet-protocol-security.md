## `IPsec`: Internet Protocol Security

> `IPsec`: a **suite of protocols** designed to **secure internet protocol (IP) communications**&#x20;
>
> &#x20;      by **authenticating and encrypting each IP packet in a communication session**
>
> secures data over public networks&#x20;
