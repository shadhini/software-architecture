# SSL: Secure Sockets Layer

## `SSL`: Secure Sockets Layer

> the predecessor to TLS

* serves as legacy encryption protocol

{% hint style="info" %}
`TLS` has **mostly replaced** `SSL` due to vulnerabilities.
{% endhint %}



### Mode of Operation

* operates at the transport layer
* establish a secure connection by encrypting data during transmission; between client and server
* uses certificates to authenticate the server and initiate an encrypted session&#x20;



### Application

* was used for securing websites and online transactions, which no use TLS

{% hint style="danger" %}
Not recommended to use now.
{% endhint %}

### Advantages

* provides secure communication for legacy systems



### Limitations

* vulnerable to security attacks
