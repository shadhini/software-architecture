# Programming Languages

{% hint style="info" %}
Computers use **machine code**.

Machines understand binary code; **1s and 0s**.
{% endhint %}

{% hint style="info" %}
Programming languages are in **human-readable code**.
{% endhint %}



## Programming Language Levels&#x20;

1. **`High-Level`** programming languages
   * more sophisticated
   * used common English
   * human-readable
   * one statement can be translated into multiple machine code instructions
   * E.g:
     * **Query Languages**: `SQL: Structured Query Language`
     * **Structured Programming Languages**: `Pascal`&#x20;
     * **Object-Oriented Programming Languages**: `Python`
2. **`Low-Level`** programming languages
   * use simple symbols to represent machine code, 0s and 1s
   * simple readable format
   * entered one line at a time â€” with one statement per line
   * one statement translates into just one machine code instruction
   * e.g:
     * **Assembly Languages**: `ARM`, `MIPS`, `X86`
