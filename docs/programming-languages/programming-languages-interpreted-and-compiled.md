## Interpreter

> **Interpreter**
>
> read & execute the source code line by line
>
> * takes the human-readable scripted source code
> * translates it into machine code
>
> built into&#x20;
>
> * web browser, or &#x20;
> * operating system of the device (require a program installed on the device to translate the code)

## Interpreted Programming Languages

* **`JavaScript`**:&#x20;
  * simpler scripting language that runs through the **web browser interpreter**
    * runs on the client side in web browsers
  * case-sensitive
  * simple scripts are run with HTML
  * complex scripts are run in separate files&#x20;
  * should not be confused with Java
* **`Python`**:&#x20;
  * scripting language
  * general-use
  * case-sensitive
  * used&#x20;
    * with `Windows`, `macOS` & `Linux` operating systems&#x20;
    * with server-side web app code
  * requires `Python engine` to interpret code
  * easy to learn & use for developers
* **`HTML`**: Hypertext Markup Language
  * mostly case-insensitive with some exceptions
  * uses tags to format web pages on client-side web browsers
* **`Lua`**: general-purpose, lightweight game scripting language&#x20;
  * easy to learn & use for developers



## Compiled Programming Languages



* **`C`**, **`C++`**, **`C#`**
  * `C`: original language & `C++`/`C#`: variations of `C`
  * case sensitive&#x20;
  * used in many operating systems like `Microsoft Windows`, `Apple macOS`, and `Linux`
  * takes more time to learn and use for coding&#x20;
  * required less memory&#x20;
  * code runs faster
* **`Java`**&#x20;
  * case-sensitive
  * object-oriented programming language
  * requires **`JVM`**: `Java Virtual Machine` to run the code&#x20;
  * **cross-platform** language: works well across platforms
    * runs the same code on many platforms (macOS, Windows, Linux)
  * `Android OS` is written in Java
