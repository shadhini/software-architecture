# Software Stacks

## Software/Technology Stack

### Software Stack

> **— combination of technologies —**&#x20;
>
> * set of individual technologies stacked in a hierarchy to support the execution of an application from user to  computer hardware

typically include

* **front-end technologies**
  * programming languages
  * frameworks
  * UI tools
* **back-end technologies**
  * programming languages
  * frameworks
  * web servers
  * app servers
  * operating systems
  * messaging applications
  * databases



#### Parts of Software Stack

* you can use only the layers you need for your application



simplest stack

1. Presentation  layer
2. Logic layer
3. Data layer



complex stack

1. Presentation  layer
2. Logic layer
3. Data layer
4. Security
5. Virtualization
6. Orchestration

* scheduling & orchestration
* runtime environments
* database connectivity
* networking&#x20;

#### Software can be from different sources

* Internal sources
* Third party software
* Cloud providers



### Software Stacks

#### Python-Django stack

> * **`Python`**: programming language
> * **`Django`**: web framework

* open-source&#x20;
* commonly used for&#x20;
  * large-scale, fast-changing web applications

#### Ruby on Rails Stack

> * **`Ruby`**:  programming language
> * **`Ruby on Rails`**: server-side web application framework

* great with
  * **`JSON`** or **`XML`** for data transfer&#x20;
  * **`HTML`**, **`CSS`** and **`JavaScript`** for front-end web development



#### ASP.NET Stack

> **Microsoft** technologies
>
> * **`ASP.NET MVC`** framework
> * **`IIS`** web server
> * **`SQL`** Server&#x20;
> * **`Azure`**



#### `LAMP` Stack&#x20;

> * runs on **`Linux`** OS
> * **`Apache`**&#x20;
>   * **`HTTP`** or **`Web`** server
> * **`MySQL`** database
> * **`PHP`** programming language

* early example
* open source
* loosely coupled - easy to swap options
  * **`LAPP`** with **PostgreSQL** instead of MySQL
  * **Python** or **Perl** instead of PHP
* designed for building websites & cloud applications
* front-end uses JavaScript & HTML





#### `MEAN` Stack

> * **`MongoDB`** database
> * **`Express.js`** web application server framework
> * **`Angular.js`** framework for front end JavaScript development
> * **`Node.js`** platform for server side scripting

MEAN software & services are

* platform agnostic&#x20;
* free & open-source

#### `MERN` Stack

> Replaces `Angular.js` with **`React`** in **MEAN** stack

* flexible & high performing framework for developing front ends&#x20;

#### `MEVN` Stack

> Replaces `Angular.js` with **`Vue.js`** in **MEAN** stack

* **`Vue.js`**:&#x20;
  * lighter weight JavaScript framework with less capabilities
  * provide better performance than `Anjular.js`
