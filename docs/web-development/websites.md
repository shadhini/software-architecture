# Websites

## Interaction with a Website

<figure><img src="../.gitbook/assets/website-interations.svg" alt="" width="563"><figcaption></figcaption></figure>

* launch an internet browser: Google Chrome, Microsoft Edge, Mozilla Firefox, Apple Safari
* enter URL to the browser address bar
  * URL: www.ibm.com
* browser contacts the server with name and requests the information that makes up the website
* before the sever sends the code and data to the client
  * back-end
    * processes the data that user entered while browsing
      * login information
      * product searches
      * user payment info
* **server** sends a **response** which contains the data that the client requires to display the website&#x20;
  * **`HTML`**: defines structure  & content of the page
  * **`CSS`**: adds style and flair to the page
  * **`JavaScript`**: adds interactivity and dynamic content
